# Public Positioning

## Short Description

Open-source infrastructure for ethical menstrual health research.

## Longer Description

Menstrual Health Open provides schemas, synthetic datasets, a basic validation
baseline, governance resources, and open-ended project inspirations for building
responsible menstrual health data infrastructure. The project is open-source,
but real participant-level health data is protected and not publicly released.

## Suggested Public Copy

This project is an open-source infrastructure initiative for ethical menstrual
health research. The code, schemas, documentation, synthetic datasets, research
methods, and governance resources are open for public contribution. Sensitive
participant-level data collected through global partners is protected and not
publicly released. Approved researchers may request controlled access under the
project's data governance and ethical review process.

## Avoid Saying

- "Open menstrual health database"
- "Public raw menstrual health data"
- "Upload real participant data for the hackathon"
- "Clinically validated diagnostic platform"
- "Medical device"

## Prefer Saying

- "Open-source infrastructure"
- "Synthetic datasets for development"
- "Privacy-preserving research tools"
- "Ethical data governance"
- "Diagnostic-support prototypes"
- "Aggregate insights after privacy review"
