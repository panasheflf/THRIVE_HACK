# governance/consent_templates/

Informed consent templates for menstrual health research.

Public prototypes should use synthetic or fictional data by default. Do not use
this template as a shortcut for collecting real participant data; real
collection requires ethics review, local adaptation, participant-language
materials, and a documented data protection plan.

## Files

| File | Purpose |
|------|---------|
| `informed_consent.md` | Complete informed consent form template |

## What the Template Covers

- Study description and purpose
- What participants will be asked to do
- Voluntary participation and right to withdraw
- Risk and benefit disclosure
- Data protection and storage explanation
- Tiered data access model explanation
- Participant rights (access, correction, deletion, withdrawal)
- Audio/video recording consent
- AI-assisted transcription, translation, summarization, or analysis consent
- Contact information for questions and complaints

## Adapting the Template

This template should be:
1. Translated into all participant languages
2. Reviewed by your institutional ethics board (IRB)
3. Adapted to comply with local regulations
4. Tested with target communities for cultural sensitivity
5. Approved before any data collection begins

## Required Customization

Replace all `[insert here]` placeholders with your specific information before use.

## Public Demo Rule

Public demos, screenshots, tests, and sample uploads should use
synthetic, fictional, or volunteer test data that does not include real
participant health records or identifiable menstrual health experiences.
